##### 1.常用命令

```
hexo s  //本地启动hexo命令
hexo new xxx //新建文章;默认source/_posts/xxx.md
hexo new page cc //在source目录下新建文件夹cc,其中默认index.md ,访问其路径http://localhost:4000/cc
hexo clean && hexo g && hexo d  //hexo发布git命令
```



给文章添加分类和标签

```
title: 在这里
date: 2018-08-02 11:41:10
tags:
- 博客           //多个标签可以这样添加
- hexo
categories: web前端
```
